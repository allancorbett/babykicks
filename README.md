# Baby Kicks

A simple web app to track baby kicks during pregnancy. Tap a button, record the time, spot patterns.

**Live:** [babykicks.vercel.app](https://babykicks.vercel.app)

## Features

- **One-tap recording** — press the button each time you feel a kick
- **Add past kicks** — log kicks you missed with a date/time picker
- **Scatter plot** — shows kick times over the last 14 days to spot patterns
- **Kick log** — full history grouped by day, with delete support
- **Export/Import CSV** — back up your data or transfer between devices
- **No backend** — all data stored in your browser's localStorage

## Privacy

No data leaves your device. There are no analytics, cookies, accounts, or tracking of any kind. Your kick data lives entirely in your browser's localStorage.

## Deploy

This is a single static HTML file. To deploy on Vercel:

1. Fork or clone this repo
2. Connect it to [Vercel](https://vercel.com)
3. Deploy — no build step needed

Or just open `index.html` in any browser.
